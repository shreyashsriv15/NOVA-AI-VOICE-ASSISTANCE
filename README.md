# Nova AI Voice Assistant

An AI-powered voice assistant built using Python and Gemini AI.

## Features

- Wake-word activation ("Nova")
- AI conversations using Gemini
- Weather updates
- News headlines
- Music playback
- Website automation
- Voice responses using Edge-TTS
- Exit command

## Technologies Used

- Python
- Gemini API
- SpeechRecognition
- Edge-TTS
- OpenWeather API
- News API
- Pygame

## Future Improvements

- Open desktop applications
- Email integration
- Reminder system
- Smart home controls
